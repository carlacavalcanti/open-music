OPEN MUSIC

Aplicação front-end que consome uma API para exibição dinâmica de álbuns musicais, com filtros interativos por gênero e faixa de preço.

VISÃO GERAL

O Open Music é um projeto desenvolvido com foco em prática de JavaScript moderno e consumo de API.

A aplicação realiza a requisição de dados externos para exibir álbuns musicais dinamicamente na interface. O usuário pode aplicar filtros por gênero e faixa de preço, atualizando os resultados em tempo real, sem recarregar a página.

O projeto não possui funcionalidade de reprodução de músicas.
O foco está na manipulação de dados e construção da interface.

FUNCIONALIDADES

- Consumo de API utilizando fetch

- Renderização dinâmica de álbuns

- Filtro por gênero musical

- Filtro por faixa de preço

- Atualização dinâmica da interface

- Layout responsivo
